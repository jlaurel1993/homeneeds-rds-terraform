# HomeNeeds RDS & Terraform Infrastructure

This repository contains the database schema and cloud infrastructure
for the HomeNeedsService platform, built using AWS RDS and Terraform.

## Tech Stack
- AWS RDS (MySQL)
- Terraform (Infrastructure as Code)
- SQL
- Linux (WSL)
- Git & GitHub

## Features
- Relational database schema design
- Infrastructure as Code for AWS resources
- Secure handling of credentials and state files
- Cloud-ready deployment architecture

## Files
- `main.tf` – RDS and networking resources
- `variables.tf` – configurable infrastructure variables
- `outputs.tf` – exposed resource outputs
- `homeneeds.sql` – database schema and sample data
